```

MAX value of N                       Time complexity
   10^8                              O(N) Border case

   10^6                              O(N) Perfect
   10^5                              O(N * logN)
   10^3                              O(N ^ 2)
   10^2                              O(N ^ 3)
   10^9                              O(logN) or Sqrt(N)
```
